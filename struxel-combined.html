How it works
01
Sensor ingestion
Temperature, pressure and vibration data streamed continuously using Weibull and LogNormal simulation models per component.
Real-time · IoT
02
Anomaly detection
Isolation Forest flags statistical outliers across all three sensor dimensions. Red dots mark each detected anomaly on the telemetry chart.
IsolationForest · ML
03
Rs risk scoring
Each component receives a composite Rs score weighted by failure probability, network centrality and downstream impact.
Rs = 0.40·Pf + 0.35·C + 0.25·I
04
Recommendation
Causal chain analysis identifies upstream triggers and generates P1/P2/P3 prioritised maintenance actions with ₹ cost estimates.
Causal chain · Priority
